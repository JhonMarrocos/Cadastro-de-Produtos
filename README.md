# Cadastro de Produtos (Python)

Projeto simples feito em Python para praticar o que estou aprendendo no curso de ADS (Faculdade Estácio).

É um sistema de cadastro de produtos que roda no terminal. Dá pra cadastrar produto, remover, ver o estoque e colocar preço. Os dados ficam salvos em um arquivo `produtos.json`, então não se perde quando fecha o programa.

## O que ele faz

- Cadastrar produto (nome e quantidade)
- Remover produto do estoque
- Ver os produtos cadastrados
- Colocar/atualizar o preço de um produto

## Tecnologias usadas

- Python
- Biblioteca `json` (pra salvar os dados)
- Biblioteca `os` e `platform` (pra funcionar no Windows e no Linux)

## Como rodar

```
python cadastro_produtos.py
```

Não precisa instalar nada, só ter o Python instalado.
